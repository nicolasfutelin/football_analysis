# Football System Analysis Project

## Requirements
To run this project, you need to have the following requirements installed:
- Python 3.x
- ultralytics
- supervision
- OpenCV
- NumPy
- Matplotlib
- Pandas

## Trained Models
- [Trained Yolo v5](https://drive.google.com/file/d/1YFrQi5MDUvHxR4KseS_OGzpD_nfaADj2/view?usp=sharing)

## Execution
Download the Trained Model (best.pt) and place it in the models folder. Execute the main script from your terminal.
