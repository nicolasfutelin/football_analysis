# Football System Analysis Project

## Requirements
To run this project, you need to have the following requirements installed:
- Python 3.x
- ultralytics
- supervision
- OpenCV
- NumPy
- Matplotlib
- Pandas
line of code to run from your terminal: pip install ultralytics supervision opencv-python numpy matplotlib pandas

## Execution
Download the Sample Video and place it in the input_videos folder, download the Trained Model and place it in the models folder.
Execute the main script from your terminal: python main.py
